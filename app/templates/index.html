4. Enriquecimiento (Diccionario) y Ajustes Globales
Este valor se usa para mostrar la hora correcta en el Dashboard. RC siempre recibe UTC 0.

Esta sección permite realizar un PULL automático (ej. 1 vez al día) a un endpoint de metadatos (como /api/device/list) para construir una caché local de IMEI a Patente/Placa. Luego, durante la telemetría, el Motor buscará el IMEI e inyectará la Placa al vuelo.
