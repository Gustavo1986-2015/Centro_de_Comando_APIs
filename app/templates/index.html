¿Cómo funciona esto?
Esta herramienta permite simular el comportamiento de una API de terceros (como Schmitz) inyectando datos directamente a tu sistema.
  • Copia un archivo JSON real de los "Demo Payloads" y pégalo en la caja de abajo.
  • Por seguridad, los eventos disparados aquí se envían automáticamente al entorno TEST para no contaminar la base de datos productiva.
  • Revisa la vista de Logs de Auditoría y el Dashboard Principal para ver el impacto en tiempo real.
